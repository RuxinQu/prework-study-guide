HTML

  • HTML stands for HyperText Markup Language.
  • <html lang="en"></html> tells that the file is written in English.
  • <meta name="viewport" content="width=device-width, initial-scale=1"/> can scale the HTML webpage to be responsive to different screen sizes.
  • Most elements need a open tag and close tag.
    Note: img is a self-close element <img src=""/>
  • <header><nav><session><main><footer> and other semantic elements can make HTML file easy to read.

CSS

  • CSS stands for Cascading Style Sheet.
  • Use <link rel="stylesheet" href="file path"> to link the CSS file and HTML file.
  • CSS box model:
  • Box-shadow(offset-x offset-y blur color) property can add rectangle shadow to an element.
    Filter: drop-shadow(offset-x offset-y blur color) property can add shadow to different shapes of element.
  • When the width and height are the same for an image, img{border-radius: 50%;} can set the image to round.
  • Flexbox can make it easier to allign flex items.

Git

  • Create a repo on Github or fork other's repo.
  • Git clone the SSH link to local computer.
  • Create a new branch using git checkout -b branchname.
  • Make changes on this branch, then git add it to the staging area then git commit -m " ".
  • Run git pull origin main to always make sure the branch is up to date.
  • Git push the branch to the remote.
  • Create a pull request on Github.
  • If the pull request is approved, it can be merged to the main branch.
    (Difference between Git and Github: Git is a version control system while Github is a hosting service allows people to do version control.)

JavaScript

  • Use <script src='file path'></script> to link the HTML file with JavaScript file.
  • Use var variable to create a variable. To declare a function, using function functionname().
    Note: JavaScript is case-sensitive.
  • Use = to assign a value and === to test if two values are equal.
  • Main types of data: string, number, Null, boolean, undefined and object.
  • The method console.log() can be used to debug. The result can be seen in Chrome DevTools.
  • Array can contain a group of data of multiple types. Index is the location of an element in the array, starting from [0].
  • A for loop contains: starting point (var x = 0), condition (x < array.length), move on to the next one(x++).
  • Falsy value: 0, empty string, Null(no value), undefined(declared variable lacks a value), NaN.
