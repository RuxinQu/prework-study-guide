Git Workflow
  1. Create a repo on Github or fork other's repo
  2. Git clone the SSH link to local computer
  3. Create a new branch using git checkout -b branchname
  4. Make changes on this branch, then git add it to the staging area then git commit -m " "
  5. Run git pull origin main to always make sure the branch is up to date
  6. Git push the branch to the remote
  7. Create a pull request on Github
  8. If the pull request is approved, it can be merged to main branch
